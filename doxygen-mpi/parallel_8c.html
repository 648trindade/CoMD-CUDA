Detailed Description

Wrappers for MPI functions.

This should be the only compilation unit in the code that directly calls MPI functions. To build a pure serial version of the code with no MPI, do not define DO_MPI. If DO_MPI is not defined then all MPI functionality is replaced with equivalent single task behavior.
